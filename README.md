# Daily-calories-tracker-
A simple Python-based Daily Calorie Tracker that helps users log meals, calculate total &amp; average calories, and check if they are within their daily calorie limit.

Project Overview

This is a beginner-friendly Python program that helps users track their daily calorie intake.
It allows the user to:
Enter multiple meals and their calorie values
Calculate total and average calories
Compare the result with a daily calorie limit
Display a formatted summary table in the terminal
